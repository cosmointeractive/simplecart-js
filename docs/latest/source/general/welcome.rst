
Welcome
=======

Welcome to `Page Studio CMS <http://pagestudiocms.com/>`_ an open source lightweight Content Management System (CMS) built on `CodeIgniter <http://codeigniter.com>`_.

Features
########

-  Open Source (built on Codeigniter)
-  Easy to use
-  Quick & Easy Setup
-  Inline Editing
-  Google Analytics
-  Multilingual sites
-  Content caching
-  Content revisions
