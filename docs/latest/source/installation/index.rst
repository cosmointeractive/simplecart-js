Server Requirements
###################

- PHP version 5.4.1 or newer
- MySQL 5.0+
- A database supported by the CodeIgniter Framework

.. note:: Page Studio is only tested in a Linux, Apache, MySQL, PHP environment.

Installation Instructions
#########################

Get the latest version of PageStudio from http://pagestudiocms.com

Unzip and upload the directory's contents to where you will be installing PageStudio.

Ensure that the following files and directories have writable permissions:

- assets/cms/cache/
- assets/cms/uploads/
- assets/cms/image-cache/
- application/config/config.php
- application/config/database.php

Open your web browser and go to http://yourdomain.com/install and follow the on screen instructions.

Once PageStudio has installed successfully delete the "install" folder from the root directory.
