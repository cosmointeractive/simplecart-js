Usage

Embeds a theme partial into a theme layout, content type, or entry. Theme partials are located in the partials directory of the current theme's directory (/themes/theme_name/views/partials).

{{ theme:partial name="name_of_partial" }}

Parameters

    name= *Required

name= *Required

The filename without the extension of the theme partial to load.

name="name_of_theme_partial"
