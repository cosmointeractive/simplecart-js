
Entry Name Tag
==============

.. contents::
   :local:
   :depth: 1

Introduction
#####

Returns the title of the entry ID specified.

.. code-block:: php 

    {{ content:entry_name entry_id="25" }}

Parameters
##########

.. contents::
		:local:

entry_id=
---------

The #ID of the entry of which to return the title.

``entry_id="15"``
