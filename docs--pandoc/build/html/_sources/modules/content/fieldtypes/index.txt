
Field Types
==============

.. contents::
   :local:

Introduction
#####

Fieldtypes are what allow you to enter specific types of information and provide an intuitive user interface for content entry. PageStudioCMS comes with several pre-installed fieldtypes to cover the most common types of content.

Available Field Types
############

The following field types are available with the default install of PageStudioCMS:
