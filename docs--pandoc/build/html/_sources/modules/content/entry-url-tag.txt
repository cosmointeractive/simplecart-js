
Entry URL Tag
=============

.. contents::
   :local:
   :depth: 1

Introduction
#####

Returns the full URL of the entry ID specified.

.. code-block:: php 

    {{ content:entry_url entry_id="25" }}

Parameters
##########

.. contents::
		:local:

entry_id=
---------

The #ID of the entry to return the full URL of.

``entry_id="15"``
