
Content Module
==============

This is by far the largest and most feature rich module of them all. The content module is made up of entries, content types, content fields, categories, and code snippets.

It will take me a while to document all of the GUI for this module so please keep checking back. Until then, check out the Content Plugin for example usage in your themes.
