
Users Module
============

.. contents::
   :local:
   :depth: 2

The users module is where you manage your site's users. To start, go to Users > Manage Users. You'll see a paginated list of your current users. 

You can search and filter them using the filter tools. Typing into the text box will search names and usernames of users.
